# ✅ 第1周课程安排（智能体认知 + 平台基础）

### 🎯 本周目标：建立对AI智能体的全面认知，掌握基本概念、就业路径及平台操作，完成第一个智能体Demo。

------

## 📅 周一：智能体基础认知与岗位前景

- **上午（3h）**
  - 开班动员 + 行业趋势讲解
  - 什么是AI智能体？与大模型、ChatGPT、RPA、Prompt的区别
  - 智能体的三要素：大模型 + 工具 + 环境感知
- **下午（3h）**
  - 智能体的典型类型：对话型、任务型、流程型、角色型
  - 企业应用案例拆解：客服助手、面试官、销售代表、写作助理等
  - 就业岗位介绍：开发岗、产品岗、运营岗、Prompt工程师

------

## 📅 周二：AI Agent开发思维与核心框架认知

- **上午（3h）**
  - 智能体核心流程认知：意图识别 → 状态管理 → 动作执行 → 回应生成
  - 智能体背后的工作流程：思维链 + 多轮对话 + 数据接口调用
  - 构建智能体的底层逻辑图
- **下午（3h）**
  - Coze、Dify、文心、元器平台能力图谱对比
  - 案例展示：AI面试官是怎么做出来的？
  - 平台账号注册与环境配置准备

------

## 📅 周三：Coze平台入门实操 + 构建第一个智能体

- **上午（3h）**
  - Coze平台界面解析：Bot管理、知识、插件、工作流、发布等模块讲解
  - 快速体验：创建一个简单智能体对话（欢迎语 + FAQ）
- **下午（3h）**
  - 实战：构建一个“智能体课程助教”机器人（带知识库）
  - 调试流程：如何看日志？怎么处理异常？如何优化响应？
  - 课后任务：每人完成一个自定义对话Bot，第二天展示反馈

------

## 📅 周四：Dify平台部署与ChatFlow逻辑构建

- **上午（3h）**
  - Dify介绍 + 安装部署讲解（含Docker / 一键部署 / 远程体验）
  - 项目结构讲解（应用管理、知识库、插件、工作流）
- **下午（3h）**
  - ChatFlow可视化构建实战：输入节点 → LLM节点 → 输出节点
  - 实战演练：构建一个“新生引导智能体”
  - 平台数据结构讲解：变量、记忆、提示模板、系统提示

------

## 📅 周五：平台能力横向对比 + 第一阶段展示

- **上午（3h）**
  - Coze与Dify平台横向比较：优势、适用场景、部署方式
  - 文心智能体平台结构初探（账号注册 + 体验环境）
- **下午（3h）**
  - 小组展示：本周各自完成的智能体Demo（答题助手、课程助手、迎新机器人等）
  - 教师点评 + 技术问题答疑
  - 拓展预告：下周进入智能体结构、流程控制与对话逻辑训练

------

📌 **每日作业建议：**

- 每天下午留出30分钟练习时间 + 作业布置
- 每日课后提交练习链接（Coze Bot / Dify App）

# ✅ 第3周课程安排（插件调用 + 知识库接入 + 数据智能）

### 🎯 本周目标：学习插件调用与知识库接入，提升智能体的信息检索和数据交互能力。

- 掌握插件（API）接入、配置与调试流程
- 掌握多种知识库接入方式（文档、网页、数据库）
- 能够构建具备信息检索与数据问答能力的智能体
- 为后续复杂任务型、业务型智能体开发打下基础

------

## 📅 周一：插件（API）机制与智能体扩展思维

- **上午（3h）**
  - 什么是插件？为什么智能体必须要具备调用能力？
  - 插件工作原理解析：输入 → API调用 → 响应解析 → 输出
  - 常见插件应用场景：天气查询、搜索引擎、表单提交、数据库调用
- **下午（3h）**
  - Coze插件中心介绍与调用演示
  - 第一个插件接入实操：天气查询助手（调用第三方天气API）
  - 错误调试与响应优化技巧（JSON结构、字段映射、Fallback）

------

## 📅 周二：插件实战与多API流程设计

- **上午（3h）**
  - 进阶插件配置参数详解（Headers、Token、路径变量）
  - 多插件组合调用设计：先查天气 → 再推荐出行装备
  - 场景设计逻辑训练：如何做信息链设计
- **下午（3h）**
  - 实操项目：创建一个“出行小助手智能体”
    - 1. 获取用户城市 → 2. 查天气 → 3. 推荐出行建议
  - 教师引导：插件响应结构分析 + 多插件路径配置

------

## 📅 周三：知识库基本原理与常用结构

- **上午（3h）**
  - 什么是知识库？结构化 vs 非结构化知识管理
  - 文档知识库 vs 网页知识库 vs 表格知识库
  - 检索问答原理：向量化 → 相似度匹配 → 回答生成
- **下午（3h）**
  - Coze知识模块功能演示
  - 实操：上传“学员手册.pdf”并构建文档问答Bot
  - 知识块测试技巧：分段设计、意图精准匹配、召回优化

------

## 📅 周四：多数据源知识库集成与检索增强

- **上午（3h）**
  - Dify中构建多文档知识库 + 向量化配置流程
  - 数据安全与隐私处理原则（脱敏、格式清洗）
  - 结构化数据知识库接入（CSV、表格、数据库API）
- **下午（3h）**
  - 实操项目：构建一个“企业内训资料问答Bot”
    - 包含PDF手册、Excel课表、网页链接
  - 组合检索逻辑设计：内容层级管理 + 多源回答整合

------

## 📅 周五：插件 + 知识库综合实战演练

- **上午（3h）**
  - 小组任务：基于某一个“企业应用场景”设计完整智能体（如人事问答、社保政策查询、产品推荐）
  - 要求同时接入1个API插件 + 1个文档知识库
  - 提交内容：Bot链接 + 设计逻辑图 + 使用说明书
- **下午（3h）**
  - 各组项目展示（每组10分钟演示+教师点评）
  - 共性问题分析：接口稳定性、召回不准、知识库碎片化等
  - 教师总结 + 下周预告：将进入“任务型、流程型智能体设计”

------

📌 **每日练习任务建议：**

- 插件调用练习：配置3个API（天气、翻译、搜索）
- 知识库构建练习：上传3种文件（PDF、网页、表格）
- 每天下午30分钟总结反馈 + 问题打卡提交
